# Pose Compare App

A Flutter-based mobile application that compares user poses with reference images.

## Features

- Upload a reference pose image
- Display selected image
- Multi-page app structure (Home, Upload, Camera, Result, History)
- Clean UI design using Material 3

## Tech Stack

- Flutter
- Dart
- image_picker (for image selection)
